# 驾考全科工具

这是一个移动端友好的离线网页小工具，重点不是只刷题，而是先建立“科目一、科目二、科目三、科目四”的完整学习体系，再用训练题检查掌握情况。

## 内容

- 首页：全科知识地图，说明每科要学什么、为什么学、怎么检测。
- 科目一：法规、交通信号、道路通行、速度距离、灯光停车、违法记分、事故处理。
- 科目二：上车固定流程、离合车速、倒车入库、侧方停车、曲线行驶、直角转弯、坡道项目、扣分高发点。
- 科目三：上车准备、直线行驶、变更车道、加减挡、路口、人行横道、对向来车、超车、掉头、靠边停车、夜间灯光。
- 科目四：安全文明驾驶、恶劣天气、高速隧道、紧急避险、事故救护、盲区和特殊车辆。
- 训练：科目一/四刷题，科目二/三流程检查题。
- 复盘：错题本、收藏题、已掌握、学习统计。

## 使用

直接双击 `index.html` 打开即可。进度保存在当前浏览器的本地存储中。

部署到 GitHub Pages 后，可以直接用手机浏览器打开仓库页面链接。

## 题库扩展

页面里的“导入题库”支持 JSON 数组。每道题建议使用下面的字段：

```json
{
  "id": "custom-001",
  "subject": "科目一",
  "chapter": "道路通行规则",
  "type": "single",
  "question": "题干",
  "options": ["A", "B", "C", "D"],
  "answer": [0],
  "explanation": "解析",
  "memory": "速记"
}
```

`type` 可用 `single`、`judge`、`multi`。

`subject` 可用 `科目一`、`科目二`、`科目三`、`科目四`。导入时会自动去重合并。

## 依据

知识体系按公开驾考模块整理，并参考：

- 《机动车驾驶证申领和使用规定》
- 《道路交通安全违法行为记分管理办法》
- GA 1026-2022《机动车驾驶人考试内容和方法》

不同地区、不同考场、不同教练车的点位要求可能不同，科目二和科目三上车操作以当地考场与教练要求为准。
